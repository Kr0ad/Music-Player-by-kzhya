Music Player by kzhya

TO RUN THE PROGRAM, YOU MUST PUT MP3 FILES ONLY ON THE SONGS FILE!\
YOU MUST HAVE JDK INSTALLED ON YOUR WINDOWS COMPUTER!!!

Steps on how to run:
Open Command Prompt
Redirect to the file where its located (ex. cd C:\Users\kzhya\MusicPlayer)
Paste this specific command (java -cp ".;jl1.0.1.jar" MP3Player)
Enjoy playing music!

For running on IDE's:
Open the downloaded zip file
Extract to your specific location you want
Open the extracted folder
Right-click MP3Player.java
Click "Open with..." and choose your desired IDE (IntelliJ, NetBeans, Visual Studio)

NOTE: DO NOT DELETE jl1.0.1.jar FILE AS THIS IS THE REQUIRED PROGRAM TO RUN.
